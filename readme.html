Caro investigatore,

ti ringraziamo per fornire il tuo supporto in questa indagine e affinché tu possa condurre al meglio l’analisi, ti forniamo di seguito alcune indicazioni.strong>

Prepara il tuo arsenale di tools forensi come segue:

  • Notepad o Notepad++
  • Autopsy
  • Volatility
  • Event Viewer
  • Wireshark
  • VirusTotal
  • Mitre Framework
  • Cyberchef

Nel corso dell’investigazione dovrai rilevare la presenza di un file malevolo, non è assolutamente richiesta l’analisi, pertanto non estrarlo, verrà rilevato da eventuali antivirus.

Consigliamo fortemente di prendere appunti per ogni singolo step sia per quanto riguarda le evidenze trovate, sia per le risposte corrette, il form condiviso non consente di salvare le risposte, pertanto qualora non dovessi terminare l’investigazione in una volta sola, dovrai nuovamente reinserirle!
